# Аналитика поведения игроков мобильной игры

#### Задачи проекта
1. Разработать функцию расчёта retention игроков по дням с момента регистрации.
   Данные лежат в папке shared и имеют следующую структуру:
   shared/problem1-reg_data.csv – данные о времени регистрации
   |   reg_ts  | uid  |
   |----------:|---------------|
   |  906166566|              2|
   |  906344325|              2|
   |  906686169|              2|
   |  906893386|              2|
   |  906980227|              2|

3. Проанализировать результаты A/B теста и определить более эффективный набор акционных предложений.  
4. Предложить систему метрик для оценки тематических игровых событий и оценить, как изменение игровой механики влияет на систему метрик

#### Методология и инструменты
- Python (pandas, numpy)  
- Анализ когорт (cohort analysis)  
- Статистическое сравнение групп (t-test / bootstrap / тест на независимость Хи-квадрат /  тест Д'Агостино-Пирсона для проверки нормальности данных)   
- Метрики продуктовой аналитики (ARPU, ARPPU, CR, retention)
