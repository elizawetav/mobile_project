# Аналитика поведения игроков мобильной игры

#### Задачи проекта
- Разработать функцию расчёта retention игроков по дням с момента регистрации.  
- Проанализировать результаты A/B теста и определить более эффективный набор акционных предложений.  
- Предложить систему метрик для оценки тематических игровых событий и оценить, как изменение игровой механики влияет на систему метрик

#### Методология и инструменты
- Python (pandas, numpy)  
- Анализ когорт (cohort analysis)  
- Статистическое сравнение групп (t-test / bootstrap / тест на независимость Хи-квадрат /  тест Д'Агостино-Пирсона для проверки нормальности данных)   
- Метрики продуктовой аналитики (ARPU, ARPPU, CR, retention)
